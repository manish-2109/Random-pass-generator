# 🔐 Random Password Generator

A simple and responsive Random Password Generator built using HTML, CSS, and JavaScript. This application helps users generate strong and secure passwords instantly to improve online security.

## 🚀 Live Demo

🌐 https://random-password-generator-alpha-three.vercel.app/

## ✨ Features

- Generate secure random passwords
- Custom password length
- Includes uppercase letters
- Includes lowercase letters
- Includes numbers
- Includes special characters
- One-click password generation
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📥 Installation

1. Clone the repository:

```bash
git clone https://github.com/manish-2109/Random-pass-generator.git
```

2. Open the project folder:

```bash
cd Random-pass-generator
```

3. Open `index.html` in your browser.

## 🎯 How It Works

1. Select the desired password length.
2. Choose the character types you want to include.
3. Click the **Generate Password** button.
4. Copy and use the generated password.

## 🔒 Why Use Strong Passwords?

Strong passwords help protect accounts from:

- Brute-force attacks
- Dictionary attacks
- Unauthorized access

Security guidance commonly recommends longer passwords that combine different character types and avoid predictable words or personal information. 0
